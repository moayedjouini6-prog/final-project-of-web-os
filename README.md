# WebOS

A simple Web Operating System built with HTML, CSS, and JavaScript. It recreates a desktop-like experience inside the browser with multiple built-in applications.

## Features

- 🖥️ Desktop interface
- 🧮 Calculator
- 📝 Notepad
- 🎮 Clicker Game
- 🖱️ Cursor Changer
- 📱 Responsive and easy-to-use interface

## Built With

- HTML
- CSS
- JavaScript

## How I Made It

I built this project from scratch using only HTML, CSS, and JavaScript without any frameworks. I created a desktop interface where each application opens like a window. JavaScript handles the app logic, user interactions, and window management, while HTML and CSS are used to build and style the interface.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/moayedjouini6-prog/final-project-of-web-os.git
   ```

2. Open `index.html` in your browser.
